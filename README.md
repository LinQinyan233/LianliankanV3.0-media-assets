- **音乐文件**：包含游戏内使用的MIDI文件。
- **音效文件**：包含游戏内使用的音效文件（如点击、消除、成功过关等提示音）。
- **背景图片**：提取并整理了游戏内全部背景图片。
  - **Easy（简单）**：11张，以萌宠主题为主。
  - **Normal（普通）**：11张，以风景主题为主。
  - **Hard（困难）**：11张，为千禧年风格泳装美女主题。
- **背景说明**：
  - 繁体中文原版，Hard难度的背景包含NSFW内容。
  - 流传较广的简体中文版，将Easy难度的所有背景复制用于Hard难度，以替换上述内容。
  - 传播最广的《连连看3》精简版，则删去了所有背景图片。

### 🔧 修复内容

对以下两个MIDI文件进行了修复：
- `MIDI_1028_9.mid`：修复了原文件中存在的杂音问题。
- `MIDI_1028_15.mid`：修复了左声道音量明显偏大的问题。

### 🛠️ 修复方法

上述两个MIDI文件的修复，均通过在线MIDI编辑网站 **Jampea** 完成：
1.  将需要修复的MIDI文件导入 [Jampea 在线编辑器](https://jampea.com/)。
2.  在编辑器中不做任何修改，直接使用其“导出”功能，将文件原样保存。
3.  经过此操作后，文件内置的播放参数被重置，从而解决了杂音和声道不平衡的问题。
